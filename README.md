# Usage
<!-- usage -->
```sh-session
$ npm install -g gsd-cli
$ gsd-cli COMMAND

$ gsd-cli (-v|--version|version)

$ gsd-cli --help [COMMAND]
USAGE
  $ gsd-cli COMMAND
...
```
